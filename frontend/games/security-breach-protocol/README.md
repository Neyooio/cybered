# Security Breach Protocol

An interactive branching scenario game that tests cybersecurity awareness through real-world security challenges.

## Features

- **Branching Decision Tree**: Each choice leads to different scenarios and outcomes
- **Dynamic Canvas Visualization**: Interactive particle network animation representing the cyber environment
- **Real-time Scoring**: Security score changes based on your decisions
- **10 Unique Scenarios**: Including phishing, ransomware, social engineering, and more
- **Educational Feedback**: Learn from each decision with detailed explanations
- **Multiple Endings**: Your final score determines your security expertise level

## How to Play

1. Open `index.html` in a web browser
2. Read each scenario carefully
3. Make choices based on cybersecurity best practices
4. Watch your security score and learn from the feedback
5. Navigate through branching scenarios based on your decisions
6. Reach the end to see your final security assessment

## Scenarios Include

- Phishing email detection
- USB drive security
- Password management
- Social engineering attempts
- Ransomware response
- Public Wi-Fi security
- Incident response
- Data breach handling
- Access control
- And more...

## Technologies Used

- HTML5 Canvas for dynamic visualizations
- Vanilla JavaScript for game logic
- CSS3 for modern styling and animations
- Responsive design for all screen sizes

## Learning Outcomes

Players will learn about:
- Identifying phishing attempts
- Proper incident response procedures
- Password security best practices
- Social engineering awareness
- Network security fundamentals
- Data protection principles

## Scoring System

- Start with 100 points
- Gain points for correct security decisions
- Lose points for risky or incorrect choices
- Final score determines your expertise level:
  - 90-100: Security Expert
  - 70-89: Security Professional
  - 50-69: Security Learner
  - 1-49: Security Risk
  - 0: Security Breach

Play responsibly and learn to protect your digital environment!
